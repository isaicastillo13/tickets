# Proyecto de Simulación de Página para Compra, Venta y Reventa de Boletos

Este proyecto tiene como objetivo desarrollar una página web que permita a los usuarios comprar, vender y revender boletos para diferentes eventos. A continuación, se detallan las tecnologías utilizadas y la estructura general del proyecto.
